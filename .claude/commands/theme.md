I want to change the color theme of my portfolio. Here are the available options:

**Color Themes:**
- cyan (default) - Cyan/Teal gradient
- purple - Purple/Violet gradient
- emerald - Emerald/Green gradient
- orange - Orange/Amber gradient
- blue - Blue/Indigo gradient

**Light/Dark Mode:**
- light
- dark

Please help me update the theme. I can specify a color theme, a mode (light/dark), or both.

Examples:
- "/theme purple" - Switch to purple color theme
- "/theme dark" - Switch to dark mode
- "/theme emerald light" - Switch to emerald theme in light mode
