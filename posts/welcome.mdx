# Welcome!

Welcome to my portfolio and blog! I'm excited to share my projects and thoughts on web development with you.

## What You'll Find Here

This site showcases my work and serves as a hub for my thoughts on software development. Here's what you can expect:

### Projects

I'll be featuring my latest projects here, all automatically pulled from my GitHub profile. Each project includes:

- Live demos (when available)
- Source code links
- Technology stack details
- Project descriptions

### Blog Posts

I'll be writing about:

- Web development tutorials
- Best practices and patterns
- New technologies and frameworks
- Project case studies
- Developer tips and tricks

## Technologies Used

This portfolio is built with modern web technologies:

- **Next.js** - React framework for production
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **MDX** - Markdown with JSX support
- **GitHub API** - Dynamic project fetching
